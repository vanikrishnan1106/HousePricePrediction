# HousePricePrediction
House price prediction means using a computer to guess a home's value based on past sales.  
The Input: You tell the computer a house's details (like size, bedrooms, and location).  
The Math: It compares those details to recently sold houses nearby. 
The Output: It gives you a realistic price tag.  
House Price Prediction App

Welcome to the House Price Prediction App!

This project is a simple Machine Learning web application built using Streamlit. It allows users to estimate the price of a house by entering basic property details such as area, number of bedrooms, and number of bathrooms. The application uses a Linear Regression model trained on a synthetic housing dataset and provides instant predictions through an interactive and user-friendly interface.

 Why I Built This Project

The goal of this project was to understand the complete workflow of a Machine Learning application—from data generation and model training to deployment with Streamlit. Along the way, I also focused on creating a clean and attractive user interface to improve the overall user experience.

## Features

- Predicts house prices instantly
- Modern and responsive Streamlit interface
- Uses Machine Learning with Linear Regression
- Displays model evaluation metrics (R² Score and MAE)
- Supports Indian currency formatting (Lakhs and Crores)
- Includes input validation for better predictions
- Fast performance using Streamlit caching

## Technologies Used

- Python
- Streamlit
- NumPy
- Scikit-learn
- Joblib

## How the Model Works

The application generates a synthetic housing dataset containing:

- House area (sq.ft)
- Number of bedrooms
- Number of bathrooms
- Estimated house prices

A Linear Regression model is trained using this dataset and then used to predict prices based on user inputs.
